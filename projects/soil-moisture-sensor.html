Soil Moisture
Sensor.

ESP32-based real-time soil monitoring. Live sensor readings pushed to a locally-hosted web dashboard. IoT fundamentals applied to agriculture.

ESP32 C++ IoT Capacitive Sensor Web UI Agriculture

cat components.txt

MicrocontrollerESP32 — WiFi-enabled, analog ADC input
SensorCapacitive soil moisture sensor — analog output
PowerUSB-powered
OutputLocal web server — live readings in browser
LanguageC++ (Arduino framework), HTML/CSS for dashboard
$

cat how-it-works.txt

capacitive sensor reads soil moisture as analog voltage
ESP32 ADC converts voltage → raw moisture value (0-4095)
capacitive sensors vary between units — calibration matters
ESP32 hosts HTTP server on local network
browser connects to local IP → sees live moisture reading
dark-themed UI, updates in real time without page reload
no cloud, no app — works entirely on local WiFi network
$

cat features.txt

live data
Moisture readings update in real time on the web dashboard. No refresh needed. The ESP32 serves the page and pushes updates via the same connection.
local only
Everything runs on the local network. No cloud account, no subscription, no data leaving the device. Connect to the same WiFi, open a browser, done.
dark UI
Custom dark-themed web interface. Readable, clean, consistent with the kind of UI I'd actually want to look at.
